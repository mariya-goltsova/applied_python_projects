# Telegram Fitness Bot 🥗💧🔥

Telegram-бот для отслеживания питания, воды и физической активности пользователя.

## 🚀 Возможности
- Настройка профиля пользователя (вес, рост, возраст, активность, город)
- Подсчёт дневной нормы воды и калорий
- Логирование:
  - выпитой воды
  - съеденной еды (калории)
  - сожжённых калорий (тренировки)
- Просмотр текстового прогресса
- Построение графика прогресса по воде и калориям
- Логирование всех команд пользователя

## ⚙️ Переменные окружения
Перед запуском необходимо задать .env file:
```
BOT_TOKEN=your_api_telegram_bot
WEATHER_API_KEY=your_api_weather
```

## 🐳 Запуск в Docker
```
docker build -t tg-calories-bot .

docker run --name tg-bot --env-file .env tg-calories-bot
```
## Структура проекта
```
tg_bot_calories/
├── bot.py # Основной файл запуска бота
├── config.py # Конфигурация и настройки (токен, данные БД)
├── handlers.py # Обработчики команд и сообщений
├── middlewares.py # Промежуточное ПО (middleware)
├── states.py # Состояния FSM (Finite State Machine)
├── services.py # работа с API/расчетами
├── utils.py # Вспомогательные функции
├── requirements.txt # Зависимости проекта
├── Dockerfile # Конфигурация Docker контейнера
├── venv/ # Виртуальное окружение (исключено в .gitignore)
└── pycache/ # Кэш Python (исключено)
```
